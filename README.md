# FinTrack Pro Web

Financial management dashboard built with **Angular 21**, **PrimeNG**, and **Jest**. This project is part of the FinTrack Pro ecosystem, designed to provide a clean and efficient user experience for personal finance tracking.

## Tech Stack
- **Framework**: Angular 21
- **UI Components**: PrimeNG
- **Styling**: SCSS
- **Testing**: Jest
- **Icons**: PrimeIcons

## Getting Started
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Start the development server with `npm start`.
4. Run unit tests with `npm test`.
